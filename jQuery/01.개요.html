jQyery 개요

* jQuery란?

기존의 자바스크립트 언어만으로 구현하기엔 복잡했던 구문 작성들을
간소확 하기 위해 '존 레식'에 의해 만들어짐
javaScript 기반 라이브러리
즉, javaScript언어로 유용한 함수나 내용들이 이미 정의 되어있는 모음집

복잡한 자바스크립트 구문을 보다 짧고 쉽게 개발 가능
> jQuery의 장정
1) DOM요소와 관련된 스크립트를 보다 쉽게 구현할 수 있음
2) ajax(비동기식 통신), 이벤트 처리 등 폭넓게 지원
3) jQuery와 관련된 확장형 플러그인, 오픈 api등을 지원
(차트, 슬라이드, 등등 ...)

* jQuery 라이브러리 연결 방법

head 태그내에 script태그로 외부 jQuery관련된 파일 연결
오프라인, 온라인 방식 있음
- jQuery 공식 사이트 공식사이트

1. jQuery를 직접 다운로드 받아 경로를 지정하여 사용
2. CDN 방식

* jQuery 구문 작성하기

jQuery 구문으로 DOM요소를 다루는 구문을 기본적으로 작성하기 때문에
문서상에 요소들이 다 만들어지고나서 실행되어야 함

* 순수 자바스크립트 
		window.onload = function() {
			실행내용;
		}

		javaScript에서의 window.onload구문은

		해당 문서의 요소들이 모두 로드되고 뿐만 아니라 연동되어있는
		외부 문서들도 다 가져온 뒤에 실행됨
		이문에서 단 한번만 작성 가능
> jQuery에서는 3가지 구문으로 제공 
		1) jQuery(document).redy(function(){
			실행내용
		});

		2) $(document).redy(function(){
			실행내용
		});

		3) $(function(){
			실행내용
		});

		해당 문서의 요소들만 다 로드되면 곧바로 실행됨
		여러번 기술가능함(제한없음) 순서대로 실행
